# App Store — release notes

Per-version copy for App Store Connect → Sudoku Crew → App Store → (selected version) → **"What's New in This Version"**. Hard limit: **4000 characters**, but short is better — most users skim the box for a few seconds and bounce.

Latest at top.

---

## v1.1.2 — cleaner highlights

```
• "Highlight rules" off now means fully off — no row, column, or box tinting when a cell is selected. A completely clean board if you prefer it.
```

---

## v1.1.1 — invite deep links

```
• Tap-to-join multiplayer invites — paste a Sudoku Crew link in a message and friends open straight into the game, no codes to copy.
• Invites work on iPhone and Android from the same link (and friends without the app are guided to the right store).
```

---

## v1.1.0 — multiplayer (build 12)

```
• Multiplayer mode — turn-based async sudoku for two or more players. Each turn, place one digit; wrong placements end your turn. Push notifications poke the next player when it's their go.
• Cross-device daily history — sign in on a new device and your previously-completed dailies show up automatically.
• Faster cold start — several seconds shaved off launch.
• Cleaner leaderboard badges — three independent badges (Solo / Manual / Flawless) replace the previous tangled set. Highlighting toggles no longer affect badges.
• Tappable leaderboard rows — tap any player to see their badges and time breakdown.
• Seven more tutor techniques: XYZ-Wing, W-Wing, Skyscraper, Empty Rectangle, 2-String Kite, Finned X-Wing, Finned Swordfish (21 techniques total).
• Account deletion in Settings (no longer needs a website round-trip).
```

## v1.0.6 — first public release

```
Welcome to Sudoku Crew!

• Daily puzzle race for your friend group
• Private leaderboards — no public rankings, no strangers

• Smart difficulty calibration based on the techniques actually needed to solve
```
